DSOT_battery_metadata.json contains the mean and deviation ranges for battery population needed in gridlabd. Battery
parameters required for agent preparation are provided in default_case_config.json
-----------------------------------------------------------------------------------------------------------------
**definition:
- capacity: battery kWh capacity
- rated_charging_power: maximum (rated) power while charging
- rated_discharging_power: maximum (rated) power while discharging
- inv_efficiency: inverter efficiency in percentage
- rated_charging_loss: loss % while charging
- rated_discharging_loss: loss % while discharging
